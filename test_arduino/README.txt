This .txt file explains briefly the different files of the folder

ahrs
Not used in the project
Calculate the vertical acceleration of the IMU board - different algorithms available
Could be used to improve the fall detection algorithm

board_test
Used in the beginning of the project to get familiar with the MCU
Different files to test the Adafruit ESP32-S3 board and its peripherals

get_imu_data_BLE
Test code to send IMU data to a laptop via BLE, read from the laptop (Python) and plot 

PCB_test
Test code for the custom PCB designed to control the latching solenoid
